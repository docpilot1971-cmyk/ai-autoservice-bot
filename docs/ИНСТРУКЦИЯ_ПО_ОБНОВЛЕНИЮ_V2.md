# ИНСТРУКЦИЯ ПО ОБНОВЛЕНИЮ С V1 НА V2

## Миграция с версии 1.0.0 на 2.0.0

---

## 1. Изменения в `.env`

Добавлены новые переменные. Скопируйте их из `.env.example`:

```env
# === ОБЯЗАТЕЛЬНО ДОБАВИТЬ ===

# Путь к CA-сертификату Сбера (опционально)
# Если сертификат в системном хранилище — оставьте пустым
GIGACHAT_CA_CERT=

# TTL кеша в днях
CACHE_TTL_DAYS=7

# Лимит для keyword-поиска (защита от OOM)
KEYWORD_SEARCH_LIMIT=2000

# Включить hash-fallback для embeddings (не рекомендуется)
EMBEDDINGS_FALLBACK=false
```

---

## 2. Новые зависимости

Добавлен `pydantic>=2.0.0` в `requirements.txt`:

```bash
pip install -r requirements.txt
```

---

## 3. Изменения в API

### Эндпоинт `/api/chat/ask`

**Формат запроса** — теперь строго через Pydantic-модель:

```json
// Было (v1): любой JSON с полями query и use_cache
// Стало (v2): строгая валидация

{
  "query": "Сколько стоит замена масла?",
  "use_cache": true
}
```

**Формат ответа** — теперь через Pydantic-модель (совместим с v1):

```json
{
  "answer": "...",
  "from_cache": false,
  "meta": {
    "model": "GigaChat",
    "retrieval": "hybrid",
    "top_k": 4
  }
}
```

### Эндпоинт `/api/admin/documents/upload`

Ответ теперь через модель `UploadResponse` — добавлено поле `files_saved` (было `files_saved` в v1, совместимо).

### Эндпоинт `/api/admin/documents/tasks/{task_id}`

Ответ теперь через модель `TaskStatusResponse` — добавлены поля `result`, `error`, `started_at`, `finished_at`.

## 6. Порядок обновления

```bash
# 1. Остановите текущую версию
sudo systemctl stop avtoritet-bot

# 2. Сделайте бэкап
cp -r /opt/avtoritet-bot /opt/avtoritet-bot-backup-$(date +%Y%m%d)

# 3. Обновите код
cd /opt/avtoritet-bot
git pull origin main  # или скопируйте файлы v2

# 4. Обновите зависимости
source venv/bin/activate
pip install -r requirements.txt

# 5. Обновите .env
# Добавьте новые переменные из .env.example

# 6. Запустите
sudo systemctl start avtoritet-bot

# 7. Проверьте
curl http://localhost:8000/health
curl -X POST http://localhost:8000/api/chat/ask \
  -H "Content-Type: application/json" \
  -d '{"query": "тест"}'
```

## 7. Откат на v1

Если что-то пошло не так:

```bash
sudo systemctl stop avtoritet-bot
rm -rf /opt/avtoritet-bot
mv /opt/avtoritet-bot-backup-20260413 /opt/avtoritet-bot
sudo systemctl start avtoritet-bot
```
