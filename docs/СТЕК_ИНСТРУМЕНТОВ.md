# СТЕК ИНСТРУМЕНТОВ

## AI-ассистент для сервисного центра «Авторитет»

---

## БЭКЕНД (AI-ассистент)

| Компонент | Технология | Версия | Назначение |
|-----------|-----------|--------|------------|
| Язык | Python | 3.10+ | Основной язык разработки |
| Фреймворк API | FastAPI | 0.115+ | REST API для бота |
| Сервер | Uvicorn | 0.30+ | ASGI-сервер для запуска |
| AI-модель | GigaChat (Сбер) | — | Генерация ответов на вопросы |
| Векторная БД | ChromaDB | 0.4.18+ | Хранение и поиск по базе знаний |
| Ключевой поиск | BM25 (rank-bm25) | 0.2.2+ | Keyword-поиск (гибридный с векторным) |
| Кеш | SQLite | встроенный | Кеширование пар вопрос-ответ |
| Конфигурация | python-dotenv | 1.0+ | Загрузка переменных из .env |

### Библиотеки для обработки документов

| Библиотека | Назначение |
|------------|-----------|
| `python-docx` | Чтение .docx файлов |
| `pypdf` | Чтение PDF |
| `pandas` + `openpyxl` + `xlrd` | Чтение Excel (.xlsx, .xls) |
| `beautifulsoup4` | Парсинг HTML |
| `pytesseract` + `pdf2image` | OCR для сканов PDF (распознавание текста с картинок) |
| `Pillow` | Обработка изображений для OCR |

## API ЭНДПОИНТЫ

| Метод | Путь | Описание | Auth |
|-------|------|----------|------|
| GET | `/health` | Проверка здоровья | Нет |
| POST | `/api/chat/ask` | Вопрос к AI-ассистенту | Нет |
| POST | `/api/admin/documents/upload` | Загрузка документов в БЗ | X-Admin-Token |
| GET | `/api/admin/documents/tasks/{task_id}` | Статус задачи индексации | X-Admin-Token |

## ФОРМАТ ЗАПРОСА К БОТУ

```json
POST /api/chat/ask
{
  "query": "Сколько стоит замена масла?",
  "use_cache": true
}
```

## ФОРМАТ ОТВЕТА БОТА

```json
{
  "answer": "Замена масла — от 1000 руб. Точная стоимость зависит от марки автомобиля.",
  "from_cache": false,
  "meta": {
    "model": "GigaChat",
    "retrieval": "hybrid",
    "top_k": 4
  }
}
```
